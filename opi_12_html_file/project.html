Сборка мини-игр

Цель проекта: Разработать программу "сборку мини-игр", в которой пользователь может выбрать одну из предложенных ему игр и поиграть в них. Каждый из участников группы должен разработать отдельный модуль со своей мини-игрой.

Учасники:

Ус Егор - Модуль "Морской Бой"

Грук Егор - Модуль "Угадай число"

Борисов Антон - Модуль "Крестики-нолики"

Херунцева Дарья - Модуль "Судоку"
